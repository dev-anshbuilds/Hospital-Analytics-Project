# Hospital Resource Utilization & Patient Outcomes Dashboard

## Problem Statement
Hospitals require clear and actionable insights to manage patient flow, optimize resource utilization, monitor outcomes, and control costs across multiple departments and branches.

---

## Solution Overview
We developed an interactive analytics dashboard using Power BI, MySQL, and FastAPI that provides hospital operations teams with real-time visibility into admissions, length of stay, readmissions, bed utilization, doctor workload, patient outcomes, and financial performance.

---

## Key Capabilities
- Executive-level KPIs for hospital operations
- Department-wise performance and outcome comparison
- Resource and doctor utilization analysis
- Financial and billing insights
- Interactive filtering by branch, department, date, and demographics

---

## Technology Stack
- Power BI
- MySQL
- FastAPI
- Python

---

## Impact
The solution supports hospital administrators in identifying operational bottlenecks, improving staffing efficiency, optimizing resource allocation, enhancing patient care, and enabling data-driven capacity planning.

---

## Deliverables
- Interactive Power BI dashboard
- GitHub repository with code and documentation
- Public demo video walkthrough
